# Live Cut

This folder contains standalone browser and GitHub preview versions of
`honeymoon`.

## Directly in GitHub

Open the repository README. GitHub renders `honeymoon-github.svg` directly, so
no local command is needed for the silent visual preview.

Direct file:

```text
live/honeymoon-github.svg
```

## Browser version with audio

The optional browser cut is `honeymoon-live.html`. It is kept as an alternate
screening version with audio; the GitHub README preview above is the
no-local-run version.

Notes:

- The browser requires a click before audio can start.
- The file is self-contained: no external assets, no network, no dependencies.
- It is intended for repo viewing, screen sharing, and early live demos.
- Keep names and identifying details out of public cuts.
